CredLink is a blockchain-based academic credential verification system designed to prevent fake degrees and enable instant verification during job applications.

## Smart Contract Overview

- Allows authorized institutions to issue academic degrees to student wallet addresses.
- Stores tamper-proof degree ownership records on-chain.
- Enables read-only verification of degree ownership without document uploads.

## Repository Purpose

This repository contains an early-stage prototype smart contract created for the hackathon PPT round. The full frontend and integration will be implemented in the main hackathon round.

## Tech Stack

- Solidity
- Ethereum-compatible blockchain (Polygon Testnet)
